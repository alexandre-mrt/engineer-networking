# Engineer Networking — Project Guide

## Overview
Content-rich website about networking in tech and evolving as an engineer in the AI era. Research-driven with 200+ sourced data points.

## Stack
- HTML/CSS/JS (vanilla, no framework)
- Tailwind CSS (CDN)
- Chart.js for data visualizations
- IntersectionObserver for scroll animations
- Single-page site with smooth scroll navigation

## Structure
```
/
├── index.html          # Main site (single page)
├── src/
│   ├── css/style.css   # Custom styles (CSS variables, dark mode)
│   ├── js/main.js      # Interactions, animations, charts
│   └── assets/         # Images, icons
├── research/           # Source data (not deployed)
├── DESIGN.md           # Design system
└── CLAUDE.md           # This file
```

## Dev Commands
- Open: `open index.html` or use Live Server
- No build step needed

## Design
See DESIGN.md for full design system. Key: warm off-white backgrounds, fluid typography, scroll animations, dark/light toggle.
